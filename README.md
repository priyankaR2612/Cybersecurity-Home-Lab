 Cybersecurity Home Lab

 Overview
This project documents my beginner cybersecurity home lab built using Kali Linux, Metasploit, Burp Suite, Wireshark, OWASP Juice Shop, and Metasploitable2.

The objective of this lab was to gain hands-on experience in:
  Network scanning
  Vulnerability assessment
  Exploitation
  Web application security testing
  HTTP traffic interception
  Packet analysis



 Tools Used
 Kali Linux
 Nmap
 Metasploit Framework
 Burp Suite
 Wireshark
 OWASP Juice Shop
 Metasploitable2
 VMware Workstation



 Activities Performed

 1. Network Reconnaissance
 Performed network scanning using Nmap
 Identified open ports and services

2. Exploitation
Used Metasploit Framework against Metasploitable2
Successfully obtained root shell access

 3. Web Application Testing
 Deployed OWASP Juice Shop
 Tested login functionality and request interception

 5. Burp Suite Analysis
 Intercepted HTTP requests
 Analyzed traffic using HTTP History and Repeater

 5. Wireshark Traffic Analysis
  Captured network packets
  Inspected TCP and HTTP traffic
